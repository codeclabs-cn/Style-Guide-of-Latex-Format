\section{0 and nullptr/NULL}\label{sec:0-and-nullptr/null}
Use \mintinline[breakanywhere,bgcolor=code_bg_pro]{C++}{nullptr} for pointers, and \mintinline[breakanywhere,bgcolor=code_bg_pro]{C++}{'\0'} for chars (and not the \mintinline[breakanywhere,bgcolor=code_bg_pro]{C++}{0} literal).

For pointers (address values), use \mintinline[breakanywhere,bgcolor=code_bg_pro]{C++}{nullptr}, as this provides type-safety.

Use \mintinline[breakanywhere,bgcolor=code_bg_pro]{C++}{'\0'} for the null character. Using the correct type makes the code more readable.
